# GALANGMACALINAO_QUIZ3

## 📊 Student Data Analysis (Dockerized)

This project generates a dashboard using:
- Histogram
- Relative Frequency Histogram
- Bar Chart
- Pie Chart

---

## 🚀 How to Run

### 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/GALANGMACALINAO_QUIZ3.git

cd GALANGMACALINAO_QUIZ3

---

### 2. Run using Docker (ONE COMMAND)

docker compose up --build

---

Check the outputs folder:
