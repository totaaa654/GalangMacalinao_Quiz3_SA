# GALANGMACALINAO_QUIZ3

## 📊 Student Data Analysis (Dockerized)

This project generates a dashboard using:
- Histogram
- Relative Frequency Histogram
- Bar Chart
- Pie Chart

---

## 🚀 How to Run

### 1. Clone the repository

git clone https://github.com/totaaa654/GalangMacalinao_Quiz3_SA.git

cd GALANGMACALINAO_QUIZ3

---

### 2. Run using Docker (ONE COMMAND)

docker compose up --build
## if output only, run this directly to terminal
docker run -v ${PWD}/outputs:/app/outputs totaaa/student-dashboard
